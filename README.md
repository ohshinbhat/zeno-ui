# Zeno UI

Lean, publishable Zeno UI packages for React web and React Native.

## Packages

- `@zenoui/react`: web primitives, hosted theme provider, and pre-hydration theme script support.
- `@zenoui/react-native`: native primitives and hosted theme provider for React Native apps.

## Component Surface

Both packages expose the same five high-leverage primitives:

- `Stack`
- `Text`
- `Button`
- `Input`
- `Card`

## Verification

```bash
yarn install
yarn typecheck
yarn build:packages
yarn pack:packages
```
